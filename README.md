# Taani's World

A cozy pixel-art birthday website made as a tiny game menu for Taani.

## Open the site

Open `index.html` in a browser. No build step is required.

## Replace content

- Letters: edit `data/letters.js`
- Playlist: edit `data/playlist.js`
- Friends: edit `data/friends.js`
- Reasons: edit `data/reasons.js`
- Memory Museum: edit `data/memories.js`

## Assets

- Put screenshots, profile pictures, covers, and other friend images in the relevant `assets/` folders.
- Put real songs in `audio/` and update each `audio` field in `data/playlist.js`.
- Leave `audio` blank when no local audio file exists; the site will show the Spotify link instead.
